## flex知-s识


设为 Flex 布局以后，子元素的float、clear和vertical-align属性将失效。

    属性值：
    对flex容器
        flex-direction
        flex-wrap
        -justify-content
            1. /*  space-around:平分剩下的空间 两盒子之间间隔是 盒子与边框的距离2倍
            pace-evenly:平分剩下的空间 两盒子之间间隔相等 
            */
        align-items
        align-content

    对flex容器中的每一项
        order
        flex-grow
        flex-basis
        flex-shrink
        align-self
