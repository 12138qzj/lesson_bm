动态规划思想：
    1. 将一个大问题分解成若干个小问题，
    2. 取出每一个小问题的最优解，
    3. 每次计算都会将小问题的最优解，取出找到大问题的最优解。
小偷（打家劫问题） 思想：
每次偷窃都是在4个房子里面取最优解
``` js
    arr[0] = 0;//假设房子之前有一个房子 金额为0
    arr[1] = nums[0];//第一个房子的金额取出
``` 
