## 跨域
    浏览器的安全策略。
     
    一个域向另一域请求资源的时候。这个就存在跨域

    **同一个域** ：同一个协议（http/https），同域名，同一个端口。
    CORS(Cross-Origin Resource sharing 跨域资源共享)
    
    两个源不相同则发生跨域：判断--协议，域名，端口
